Fair Web Services Manifest

Using this web site

This site is a Fair Web Service. It respects the rights and freedoms of its users and gives them control over their data.

This site is a static web site. There is no login and no user accounts. That means there is no user data stored or tracked and there is no need for an API or data export.

The content of the site is published under a Creative Commons Attribution Share Alike license so you are free to use and share it provided you keep an attribution and share it under the same license. If you would like to see the sources or contribute to the site please look at the site's source code repository on GitHub.

The site is served as GitHub Pages so GitHub might see some data and server logs. This is covered in their terms of use.

The site can be accessed via HTTPS under https://cornelius.github.io/fairwebservices. It only is accessible via HTTP under its own domain http://fairwebservices.org because of limitation of GitHub Pages with custom domains and SSL. We recognize that this is an issue and will eventually fix it.

Advertisement Policy
